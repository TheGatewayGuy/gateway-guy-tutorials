Welcome to the official tutorial repository for [The Gateway Guy](https://www.youtube.com/@TheGatewayGuy) YouTube channel!

This repository contains hands-on tutorials, example code, and configurations that complement the video content on my channel. Each tutorial is designed to help you learn Kong Gateway, Kong Konnect, and API Gateway best practices through practical, real-world examples.

## About The Gateway Guy

The Gateway Guy is a YouTube channel dedicated to teaching API Gateway concepts, Kong Gateway implementations, and cloud-native API management strategies. Whether you're just getting started with API Gateways or looking to master advanced Kong features, this channel has something for you.

**Subscribe here:** [youtube.com/@TheGatewayGuy](https://www.youtube.com/@TheGatewayGuy)

## Repository Structure

Each tutorial in this repository is organized in its own numbered directory and corresponds to a video on the YouTube channel. Inside each tutorial folder, you'll find:
- Complete code and configuration files
- A dedicated README with step-by-step instructions
- Any additional resources needed to complete the tutorial

## How to Use These Tutorials

1. **Watch the Video**: Start by watching the corresponding video on [The Gateway Guy YouTube channel](https://www.youtube.com/@TheGatewayGuy) to understand the concepts
2. **Clone the Repository**: Clone this repo to your local machine
   ```bash
   git clone https://github.com/yourusername/gateway-guy-tutorials.git
   cd gateway-guy-tutorials
   ```
3. **Navigate to the Tutorial**: Go into the specific tutorial directory
   ```bash
   cd 001-konnect-dataplane-deployment
   ```
4. **Follow the README**: Each tutorial has its own README with detailed instructions
5. **Experiment**: Modify the configurations and experiment to deepen your understanding

## Prerequisites

General prerequisites that apply to most tutorials:
- Basic understanding of Kubernetes
- kubectl CLI installed and configured
- A Kong Konnect account (free tier available)
- Docker Desktop or access to a Kubernetes cluster

Specific prerequisites for each tutorial are listed in their individual README files.
